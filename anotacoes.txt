- Ativar ambiente (Windows/Linux)
.\venv\Script\activate
source venv/bin/activate

pip install -r requirements.txt

- Rodar app
python run.py

- Criar nova Branch
git checkout -b nome-da-branch

- Trocar/entrar numa Branch
git switch -c nome-da-branch

- Subir alterações na Branch
git add .

- Commitar na Branch
git commit -m "mensagem"

- Mesclar na Main (Acesse a Main e rode o comando)
